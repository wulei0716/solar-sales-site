# 光能仓太阳能产品独立站

这是一个可直接打开的静态销售网站模板，面向太阳能行业产品询盘。

## 包含内容

- 德业、艾罗 SolaX、固德威逆变器
- 德业、艾罗 SolaX、固德威储能电池
- 隆基、晶科、天合光伏组件和太阳能板
- 产品筛选、关键词搜索、询盘清单
- 邮件询盘草稿生成
- 响应式布局，适配桌面和手机
- 德业、艾罗 SolaX、固德威、隆基、晶科、天合产品已按官网产品页更新型号、图片、关键参数和来源链接

## 使用方式

直接在浏览器打开：

```text
/Users/mac/Documents/网站/solar-sales-site/index.html
```

上线前建议替换以下内容：

- `assets/app.js` 中的 `sales@your-domain.com`
- 产品图授权、功率段、认证、质保、库存和交期
- 公司名称、备案信息、客服邮箱、WhatsApp 或电话
- 如果需要自动同步官网数据，建议增加后端定时抓取和人工审核流程，避免前端跨域抓取失败或同步到过期信息

## 图片来源

产品卡片使用品牌官网公开产品页图片，并在页面保留官网来源链接。英雄区和服务流程仍使用 Wikimedia Commons 场景图。真实商用上线前，建议确认品牌方素材授权或替换为自有/授权产品图。
